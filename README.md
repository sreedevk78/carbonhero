# CarbonHero 🌿

CarbonHero is a gamified sustainability platform designed to help users track, understand, and reduce their carbon footprint. By combining real-time logging with AI-driven insights, CarbonHero transforms environmental responsibility into an engaging and rewarding experience.

## ✨ Key Features

- **🎮 Gamified Tracking**: Earn badges and progress through levels as you log eco-friendly activities.
- **🤖 Green AI Integration**: Powered by Groq (LLaMA 3), our specialized assistant provides sustainability tips and application guidance.
- **📊 Carbon Analytics**: Visualize your impact with detailed logging for travel, food, and energy.
- **🛡️ Secure & Hardened**: Environmentally conscious and security-focused with masked API credentials and robust fallback systems.
- **🏆 Boss Battles**: Use your eco-points to challenge the "Smog Boss" and clean up the virtual environment.

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sreedevk78/carbonhero.git
   cd carbonhero
   ```

2. **Setup Environment**:
   Copy `.env.example` to `.env` and fill in your API keys (Groq, Gemini, OpenAI).
   ```bash
   cp .env.example .env
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application**:
   ```bash
   python run.py
   ```

## 🛠️ Technology Stack

- **Backend**: Flask (Python)
- **Database**: SQLite (SQLAlchemy)
- **AI Engines**: Groq (Primary), Google Gemini, OpenAI
- **Frontend**: Glassmorphic UI with CSS & HTML5
